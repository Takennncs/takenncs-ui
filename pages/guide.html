Customization Guide

Colors

Edit CSS variables in assets/css/custom.css:

:root {
  --sidebar-width: 250px;
  --brand-color: #5b21b6;
  --primary-color: #007bff;
  --success-color: #28a745;
  --warning-color: #ffc107;
  --danger-color: #dc3545;
  --body-bg: #f9fafb;
}
Branding

To change the logo and name, edit the sidebar header in each HTML file:

<div class="sidebar-header text-center text-white py-4">
  <h4 class="mb-0"><i class="fas fa-code"></i> Your Brand</h4>
</div>

Adding New Pages

  1. Create a new HTML file in the pages/ directory
  2. Copy the basic structure from an existing page
  3. Update the navigation menu in all pages to include your new page
  4. Add your content in the main content area
Example Page Structure
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
  <title>New Page · takenncsUI</title>
  <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css">
  <link rel="stylesheet" href="../assets/css/custom.css">
</head>
<body>
<div class="d-flex" id="wrapper">
  <!-- Sidebar -->
  <nav id="sidebar" class="bg-dark">
    <!-- Navigation menu -->
  </nav>
  
  <div id="page-content-wrapper" class="w-100">
    <nav class="navbar navbar-expand-lg navbar-light bg-light border-bottom">
      <button class="btn btn-primary" id="menu-toggle"><i class="fas fa-bars"></i></button>
    </nav>
    
    <div class="container-fluid p-4">
      <!-- Your content here -->
      <h1>New Page</h1>
      <p>Your content goes here.</p>
    </div>
    
    <footer class="bg-light text-center py-3">
      <small>Made with <i class="fas fa-heart text-danger"></i> · Built on <strong>takenncsUI</strong></small>
    </footer>
  </div>
</div>

Frequently Asked Questions

Can I rename takenncsUI?

Yes! takenncsUI is MIT licensed, which means you can rename, modify, and redistribute it freely.

How do I change the sidebar width?

Edit the --sidebar-width CSS variable in assets/css/custom.css.

Is this template responsive?

Yes, takenncsUI is fully responsive and works on mobile, tablet, and desktop devices.

Can I use this for commercial projects?

Yes, the MIT license allows commercial use without restrictions.
